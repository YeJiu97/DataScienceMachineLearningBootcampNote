Instructions

1. A variable called a

2. A variable called b

e.g. If:

a = 5
b = 8

Your code should result in:

a = 8 
b = 5

You can test your code in a Jupyter notebook first and print out the values if it makes it easier.  Add your own code below the TODO. For the test to pass, don't change the existing code below.
