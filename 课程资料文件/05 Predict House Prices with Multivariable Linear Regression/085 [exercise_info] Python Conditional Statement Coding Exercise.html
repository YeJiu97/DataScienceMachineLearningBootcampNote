Instructions

Given a positive number n, determine whether if it is a prime number.

Write some code for the function is_it_prime(n) to check if the input n is a prime number, your function should return a Boolean value of True if n is prime, but False if n is not prime.

So if I call the function like this:

is_it_prime(13)

I would get an output of True, because 13 is indeed a prime number.

But if I call the function like this:

is_it_prime(24)

I would get an output of False.

REMEMBER: The definition of a prime number is: "A whole number greater than 1 that can not be made by multiplying two smaller whole numbers."

e.g. 9 is not a prime number because 3 x 3 = 9

But 19 is a prime number, because 19 is only divisible by 1 and 19.
